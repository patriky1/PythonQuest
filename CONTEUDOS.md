# Guia para envio dos conteúdos do PythonQuest

Quando você enviar novos conteúdos, use este formato para eu transformar em missões interativas:

## Tópico
Exemplo: Fundamentos / Controle de fluxo / Funções / Estruturas de dados

## Objetivo da aula
O que o aluno precisa aprender ao final da missão.

## Explicação teórica
Texto base da aula.

## Exemplos de código
Inclua os exemplos em Python.

## Perguntas desejadas
Se tiver perguntas, envie com resposta correta. Se não tiver, eu crio.

## Nível
Iniciante, intermediário ou avançado.

## Observação didática
Exemplo: explicar como se fosse para criança, para EJA, para concurso, para curso técnico etc.
